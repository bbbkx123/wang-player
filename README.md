# wang-player

这是一个开源的移动端web音乐播放器，后端使用开源项目 ```NeteaseCloudMusicApi```, 分为react版本和vue3版本， 目前只能使用react版本

## 技术栈
typescript / react / redux / better-scroll / antd-mobile

## 实现功能
1. 推荐歌单
2. 歌单播放
3. 歌曲搜索
4. 歌单详情
5. 播放动画实现（进度条，cd图旋转等）
6. 展示歌词
7. 展示歌曲评论

## 安装
```
yarn install
```

## 运行
```

```
